## Introduction

Welcome to the Chama App! This application is designed to help manage your chama (investment group) efficiently. It handles member contributions, document management, financial reporting, and group communications.

### Key Features
- Member payment submissions and tracking
- Document management system
- Financial reporting and analytics
- Real-time notifications
- Mobile-friendly interface
- Role-based access control

## Getting Started

### Account Setup
1. **Registration**
   - Visit the app's login page
   - Click "Register" to create a new account
   - Fill in your details:
     - Full Name
     - Email Address
     - Password (minimum 8 characters)
   - Verify your email address through the link sent to you

2. **First Login**
   - Log in with your email and password
   - Complete your profile information
   - Set up notification preferences

### Navigation
- **Top Bar**: Quick access to notifications and user menu
- **Side Menu**: Main navigation for all features
- **Mobile Menu**: Access via the hamburger icon (≡) on small screens

## Member Features

### Making Payments
1. **Submit a Payment**
   - Navigate to "Payments" → "Submit Payment"
   - Enter the amount in KES
   - Provide payment description
   - Enter M-PESA reference number
   - Upload M-PESA screenshot
   - Click "Submit Payment"

2. **View Payment History**
   - Go to "Payments" → "History"
   - Filter by date range or status
   - Download payment receipts
   - View payment status updates

### Document Access
1. **Viewing Documents**
   - Navigate to "Documents"
   - Browse by category or use search
   - Click to view or download
   - Check document version history

2. **Uploading Documents**
   - Select "Upload Document"
   - Choose document type
   - Add description
   - Upload file
   - Set visibility permissions

### Notifications
1. **Managing Notifications**
   - Access notification center via bell icon
   - Mark as read/unread
   - Filter by type
   - Set notification preferences

2. **Setting Preferences**
   - Go to Profile → Notification Preferences
   - Choose notification types:
     - Payment Status Updates
     - Document Uploads
     - Announcements
     - Financial Reports
   - Select delivery method (in-app/email)

## Treasurer Features

### Payment Management
1. **Payment Queue**
   - Access via "Treasurer" → "Payment Queue"
   - View pending payments
   - Verify M-PESA details
   - Approve/Reject with notes
   - Bulk process similar payments

2. **Manual Transactions**
   - Record non-member transactions
   - Categories: Expenses, Income, Investments
   - Add supporting documents
   - Set verification requirements

## Troubleshooting

### Common Issues
1. **Payment Issues**
   - Verify M-PESA reference
   - Check screenshot clarity
   - Confirm amount matches
   - Wait for processing time

2. **Document Access**
   - Check file permissions
   - Verify file format
   - Ensure stable connection
   - Clear browser cache

3. **Mobile Access**
   - Enable notifications
   - Allow camera access
   - Grant storage permissions
   - Update browser
